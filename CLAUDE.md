# ANNUAL-GAZER — Claude Code ガイド

## プロダクト概要

1年を1周で表現するアナログ時計型計器。
針の位置で「今年のどのあたりにいるか」を直感的に把握できるWebアプリ。
GitHub Pagesで単品サイトとしてリリース（v1）。

## 技術スタック

- React + TypeScript + Vite
- Tailwind CSS
- SVG描画（CSS transitionベースのアニメーション）
- 外側セグメント: `<path>` (バゲットカット八角形) + `transform: rotate()` で円周上に配置
- スペクトラム線: `<line>` を放射状に多数配置、`opacity` で点灯/消灯
- 内側目盛り: `<line>` で放射状配置（内向き）
- 7セグメントフォント: DSEG7 Classic（中央デジタル表示用）
- 状態管理: React hooks + LocalStorage
- 配信: GitHub Pages

## テーマシステム（v1.4）

- v1: **INSTRUMENT**（金属・計器・モノトーン・7セグ・インゴット）のみ
- v2予定: JEWEL（誕生石）、ZODIAC（星座）

## 基本設計

### 全体構造: 二重円（アナログ × デジタル ハイブリッド）

```
外側の円 ─── バゲットカットセグメント（31セル）
               インゴット風立体感、刻印数字
               月の日数に応じて有効セル数変動、1周させない

隙間 ──────── スペクトラムアナライザー風バー
               放射状バー群（48〜96本）、経過部分が灯る
               末尾に時刻テキスト（textPath湾曲）

内側の円 ─── アナログ計器（年の進捗）
               内向き目盛り（365日）、月ラベル（数字: 1-12）内側配置
               年進捗針1本のみ

中央 ──────── 7セグメント風デジタル表示
               年・月日・曜日・時刻・経過率・残日数
```

### 外側の円: バゲットカット・セグメント

- 31セル（バゲットカット八角形）が円弧状に並ぶ、12時方向 = 1日
- セル形状: 縦長の長方形の四隅を45°で面取りした八角形（SVG `<path>`）
- 外側（放射方向外端）のベベルが末広がり（FLARE=3px）
- **5面ファセット分割**: Top/Left/Center/Right/Bottomの5つのpathで立体感
- **動的光源**: dayAngle（24時間計の角度）に連動してファセットの明暗が変化
- スペキュラーハイライト: radialGradientの焦点が光源方向にシフト
- 各セル内に日数（1〜31）を**凹型刻印**（インタリオ）で表示
  - 3層テキスト: 影(左上,黒,blur) + ハイライト(右下,白) + 本体(黒)
  - フォント: Barlow Condensed（幅狭・工業的・DIN系）
- セル間に十分な隙間
- **モノクロトーン厳守**（オレンジ・緑などの色付け禁止）
- 3段階（半透明ガラス質感）:
  - 消灯（未来）: fill rgba(白,0.04)、ガラスの縁がかすかに
  - 点灯（過去）: fill rgba(白,0.12)、曇りガラスのように
  - 今日: fill rgba(白,0.22)、最も存在感
- 土日: **セルの明るさ**で区別（weekendBoost 1.25倍）、テキスト色は全日統一
- 下半分のセル内数字はカウンター回転で常に読める向きに

### スペクトラムアナライザー帯（隙間）

- 放射状バー群（48〜96本、プロトタイプで調整）
- 点灯バー: アクセントカラー、**グラデーション**（末尾ほど明るく、古いほど暗い）
- 消灯バー: 非表示 or 極薄（opacity 0.03）
- 外側・内側より控えめな存在感
- 点灯末尾に HH:MM:SS テキスト表示（textPath湾曲、座布団付き）
  - 上半分（270°〜90°）: 外向き
  - 下半分（90°〜270°）: 内向き（180°反転）
  - 12時付近で座布団表示
- 時刻フォーマット設定: HH:MM:SS / HH:MM / HH:MM(blink colon)

### 内側の円: アナログ計器

- **日ベース均等分割**: 365日（うるう年366日）で360°を等分
- **月ラベルは数字表示: 1, 2, 3... 12**（JAN/FEB表示はv2検討）
- 月ラベルは実際の日数に基づく角度で配置
- 目盛りは **内向き**（円の内側に向かって伸びる）
- 月ラベルは内側に配置
- 12時方向 = 1月1日（開始月は設定で変更可能）

### 年進捗針（唯一の針）

- 内側の円に配置
- 年間通日ベースで連続的に動く
- 1月1日 = 0°（12時方向）、12月31日 ≈ 359°
- 中心から内向き目盛りの手前まで伸びる
- 細く繊細なデザイン（計器の針の印象）

### 目盛り（365本・3段階）

| レベル | 対象 | 太さ | 長さ |
|---|---|---|---|
| 1 | 月の区切り（12本） | 太い | 長い |
| 2 | 10日区切り or 週区切り | 中 | 中 |
| 3 | その他の日 | 細い | 短い |

- 土日の目盛りにはアクセントカラーを適用
- 今日の位置にドット/ハイライトマーカーを表示

### 中央エリア: 7セグメント風デジタル表示

```
        2026
       03.31
        TUE
       17:27
      ─────────
       24.7%
      275 days left
```

- 数字は7セグメントフォント（DSEG7 Classic）で表示
- 消灯セグメントをごく薄く表示（opacity 0.05）→ 液晶ゴースト感
- 曜日・単位は通常サンセリフフォント
- 時刻と月日を最も大きく、年は中サイズ、経過率・残日数は小さめ
- 上段（日付・時刻）と下段（経過情報）の間にセパレータ

### 背景（検討中）

- 現在: ダミー背景（透明度確認用）。本番背景は未確定
- 方向性: 「計器の窓から空を覗く」— 背景に時間的意味を持たせる
  - 昼: 日の出〜日の入りに連動した暖色光源の移動
  - 夜: 月齢に応じた月光 + 季節の星の配置
  - 公転（年）: グラデーション中心が1年で一周
- 星座の扱い:
  - ZODIACテーマ: 星座として認識可能（1等星強調、星座線あり）
  - 他テーマ: 抽象的な星の配置として使用可能
- 技術メモ: 日の出/日の入りは緯度経度から計算、月齢は算術計算
- 背景が計器より目立つことは絶対にない

### アニメーション

- **呼吸する光**: ピボット（針の中心点）がゆっくり明滅（4〜6秒周期、opacity 0.3〜0.6）
- スペクトラムバーのグラデーション: 10分更新でシフト
- 中央時刻: 秒単位で更新（HH:MM:SS時）
- 10分更新: 針・バー・経過情報をCSS transition 0.3〜0.5秒で遷移

### レイヤー間の余白（最重要）

```
外側セグメント ─── [余白] ─── スペクトラム帯 ─── [余白] ─── 内側目盛り
```

- 各レイヤーが「浮いている」ように見えること
- 詰まった印象になるぐらいなら要素を細く・小さくする
- 余白を削ってはならない

## デザイン方針（INSTRUMENT テーマ）

5つの質感を融合:
- **無印良品**: 太さ・長さで情報階層を表現。装飾排除
- **VUメーター/McIntosh**: 細く繊細な針、シャープな目盛り、静かな緊張感
- **Bang & Olufsen**: ダーク背景に高コントラスト、十分な余白、補助情報は明度を落とす
- **KORG NTS-2 / Casio G-Shock**: スペクトラムバー、7セグ表示、デジタルとアナログの共存
- **Rolex Daytona**: バゲットカットの形状、精密にカットされた金属の塊の存在感

カラー: 背景・前景・アクセントの3色以内
バゲットカットはモノクロトーン厳守
アクセントカラーはスペクトラムバー点灯、土日の数字色、今日マーカーに使用

## NGリスト（厳守）

- **フォント**: 丸ゴシック・カジュアルフォント禁止
- **エフェクト**: ドロップシャドウ・グロー・グラデーション禁止（立体感はstroke明暗差のみ）
- **装飾**: ベゼル・外枠・装飾線禁止
- **余白**: レイヤー間の余白が詰まっている状態は禁止
- **針**: 太すぎる針禁止（太さ: 文字盤半径の1%以下）
- **セグメント**: 角丸禁止。バゲットカットは直線的カット
- **バゲット色**: モノクロトーン以外の色をセルに付けることは禁止（土日は数字色のみ）
- **スペクトラム**: バーが太すぎる・派手すぎるのは禁止
- **色**: 3色を超える配色禁止
- **アニメーション**: 派手な動き・点滅・パルス・バウンス禁止
- **背景**: 背景が計器本体より目立つことは禁止
- **全般**: 「足す」ことで解決しようとする発想を禁止
