#### 2. スペクトラムバー末尾に時刻テキスト（HH:MM:SS）追加

**基本実装:**
- スペクトラム帯の中央半径（SPEC_R_MID）に配置
- 点灯済み末尾（現在時刻の角度位置）にテキスト表示
- フォント: JetBrains Mono, weight 400, size 7.5, ACCENT色

**向き自動切替:**
- 上半分（0°〜90°, 270°〜360°）: tangentAngle = dayAngle
- 下半分（90°〜270°）: tangentAngle = dayAngle + 180°
- 90°/270°を閾値として切替

**座布団（zabuton）:**
- 時刻テキストの背後に背景色の矩形を配置
- rx=2, ry=2 で角を少し丸め
- フォーマットに応じてサイズ可変（HH:MM:SS: 44px幅、HH:MM: 32px幅）

#### 3. 秒表示の設定切替

**TimeFormat 型の導入:**
- `'HH:MM:SS'`: 秒表示あり（デフォルト）
- `'HH:MM'`: 秒表示なし
- `'HH:MM:blink'`: コロン点滅（偶数秒=表示、奇数秒=非表示）

**更新間隔:**
- 秒表示のために setInterval を1秒間隔に変更

Session 4で実装した内容に対し、指示と異なる箇所を修正。

  - 土曜: ACCENT (#c0785a、オレンジ)

- CELL_H: 26→38、CHAMFER: 3.5→4
- フォントサイズ: 8.5→11、letterSpacing: 0→1
- OUTER_R: 278→285

**ACCENT_WEEKEND (#8a9a6a) を削除。**

#### 2. 時刻テキスト: textPathによる円弧湾曲

**前回の問題:** 直線的な `<text>` + rotate で配置していた
**修正:** SVG `<textPath>` で円弧パスに沿って湾曲表示

- SPEC_R_MID（スペクトラム帯中央）を通る短い円弧パスを生成
- テキストはlit末尾の3°先（TIME_OFFSET_DEG=3）から配置
- 40°分の弧パス上にテキストが流れる

**向き自動切替:**
- 右半分（0°〜180°）: CW弧パス → テキスト外向き
- 左半分（180°〜360°）: CCW弧パス → テキスト内向き（180°反転）
- isRightHalf = dayAngle <= 180 で切替

**座布団:**
- 12時方向付近（dayAngle > 340 || < 20）のみ表示
- テキストがlit開始地点と重なるケースへの対策

**サイズ:**
- fontSize: 7.5→10、letterSpacing: 0.5→1
- スペクトラム帯幅14pxに対し、テキスト高さ約10px（71%）
